
What the platform does:
Since Online exam of IELTS is an official way of attending the exam. This platform enables you to participate mock IELTS tests 

Basic Flow:
Users land on the page > Leetcode like page with Practice category i.e. Listening, writing, Speaking. > Chooses category > Starts tests > To use the dashboard users must sign in but still can see basic result but its one time only and resets on reload. 

Nice to have: AI powered assistant for Q&A, Detailed Dashboard.
